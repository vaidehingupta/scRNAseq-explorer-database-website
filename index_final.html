About This Project TCW Laboratory · Team 14 · BF768

Overview

The scRNA-seq Explorer is an interactive web application for investigating cellular heterogeneity and genotype-associated differences in iPSC-derived Alzheimer's disease models. By integrating cell type annotation, gene expression across ~19,000 genes, and sample-level composition, this platform enables exploration of how APOE genotype influences cell population structure and transcriptional activity across major neural cell types.

Biological Background

Alzheimer’s disease is a neurodegenerative disorder characterized by cell-type-specific dysfunction in the brain. A major genetic risk factor is the Apolipoprotein E (APOE) gene, where the APOE33 genotype represents a baseline condition and APOE44 is associated with increased disease risk. To study how APOE genotype influences cellular function, this dataset uses single-cell RNA sequencing (scRNA-seq) from CRISPR-edited iPSC-derived brain cells. This controlled design allows for direct comparison of gene expression across genotypes at single-cell resolution, enabling identification of cell-type-specific differences associated with Alzheimer’s disease.

Platform Features

  • Single-cell Expression: Visualizes cell clustering via UMAP across all three genotype views (All, APOE 33, APOE 44) simultaneously — for both cell type annotation and per-gene expression. Supports violin, dot, and scatter plot views for any of the ~19,000 genes.
  • Multi-Gene Expression: Compares expression patterns across multiple selected genes using dot plots (bubble size = % expressing, color = avg log-normalized expression) or heatmaps clustered by expression level.
  • Cell Type Composition: Displays relative abundance of each cell type per sample. The merged panel aggregates both genotypes; individual genotype panels show APOE 33 or APOE 44 samples separately. Toggle which panels are visible using the genotype checkboxes.
  • Data Tables: Queryable datasets of gene expression values and cell type proportions, with searchable gene and cell type filters. Results can be downloaded as CSV.
  • Interactive Visualizations: All plots are built using Plotly, allowing users to hover over data points to view detailed information such as gene expression values, cell type labels, and sample metadata. Users can also click or double-click legend items to isolate or hide specific groups, enabling more focused comparisons across conditions. In addition, clicking legend items allows users to toggle between APOE33 and APOE44 views, an interactive feature provided by Plotly.

Sample & Replicate Information

Samples are labeled as APOE33_1.1, APOE33_1.2, APOE33_2.1, APOE33_2.2 (and similarly for APOE44), where each genotype includes data from two individuals, with two replicate samples per individual. For example, APOE33_1.1 and APOE33_1.2 come from the same individual, while APOE33_2.1 and APOE33_2.2 come from a second individual; the same structure applies to the APOE44 samples. This design allows us to compare differences between individuals while also assessing consistency within the same individual across replicates, improving the reliability of downstream analyses.

How Filters Affect Each Visualization

  • Single-cell Expression: Cell type filters restrict which cells appear across all six UMAP panels. Each column represents a genotype split (All / APOE 33 / APOE 44) automatically.
  • Multi-Gene Expression: Genotype and cell type filters restrict cells included in dot plot and heatmap calculations.
  • Cell Type Composition: The genotype checkboxes control which panels are displayed. The merged "All" panel aggregates both genotypes into a single average proportion. Cell type filters restrict which cell types appear in all panels.

Data Description

The dataset consists of processed scRNA-seq outputs including UMAP dimensionality reduction and log-normalized gene expression values for approximately 19,000 genes. All visualizations query a relational MariaDB database, enabling efficient gene-level lookups via indexed long-format expression tables.

Interpretation Notes

All expression values are log-normalized. In UMAP plots, higher color intensity indicates higher expression. In dot plots, bubble size represents the proportion of cells expressing the gene and color represents average log-normalized expression. Comparisons reflect relative rather than absolute expression levels.

Limitations

The dataset includes a limited number of samples, which may introduce sampling bias. Certain cell types may be over- or under-represented. Expression values are log-normalized and relative.
